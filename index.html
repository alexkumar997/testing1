Play Token Rule

Play Tokens are used to control server load and heavy traffic.

  • Limited tokens are added every hour
  • Every valid story play uses 1 token
  • A valid play counts only after 50% playback completion
Example:
4 AM → 200 tokens added
1 valid play → 199 remaining
Next valid play → 198 remaining

Unused tokens carry forward to the next hour.

New tokens are added every hour from 4 AM to 10 PM.

After 11 PM no new tokens are added.

At 4 AM daily reset starts a new token cycle.

If hourly tokens are exhausted:
"Hourly server limit reached. Please try again next hour."
If daily limit is exhausted:
"Daily server limit reached. Please try again next day."

This system exists to protect server stability and distribute traffic fairly across all users.
